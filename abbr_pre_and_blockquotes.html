Abbr. Pre. and Blockquotes

Abbreviations

abbr is used to mark up an abbreviation or acronym

CDGL was founded in 2020.

Blockquotes

blockquote is used to present content from another source

Codingal is on a mission to inspire all the kids in school.

Codingal

Multiple Code Lines

For multiple lines of code, use the pre element

            Codingal is on
            a mission to
            inspire all the
            kids in school

Keyboard Inputs

To indicate an input that is typically entered via the keyboard, use the kbd element

Use ctrl + p to open the Print box
